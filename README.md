# Task 2: Data Visualization and Storytelling

## Objective
Create visualizations using Power BI to convey a compelling story from the Superstore sales dataset.

## Tools Used
- Power BI Desktop
- Dataset: Superstore.csv (Kaggle)

## What I Did
- Built a 4-chart dashboard in Power BI:
  1. Category by Sales
  2. Region by Sales
  3. Category by Profit
  4. Sales by Year (trend)
- Added insight text boxes under each chart highlighting key business takeaways.
- Applied formatting (borders, clean axis, consistent fonts) to remove clutter and improve readability.
- Exported the final dashboard as a PDF report.

## Key Insights
- Technology and Furniture drive the majority of sales; Office Supplies contributes the least.
- West region leads in sales, while South lags behind.
- Furniture shows low profit despite reasonable sales, indicating possible discounting issues.
- Sales show a consistent year-over-year upward trend.
